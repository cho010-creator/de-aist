I. 연구 목표 및 주제 개요

"베타파(Beta Wave) 측정을 통해 사용자의 집중도와 능동적 정신 활동을 파악하고, 이를 기반으로 레고 스파이크 프라임(Lego Spike Prime) 로봇 팔을 실시간으로 제어하는 시스템을 구축한다."

핵심 변수: 베타파 (β Wave, 13-30 Hz)

β파는 '의도적인 사고 및 집중' 상태와 직접적인 상관관계가 있습니다. 이는 단순 이완이나 졸음이 아닌, 로봇을 능동적으로 제어하는 명령 신호로서 가장 높은 신뢰도를 가집니다.

II. 핵심 연구 방법 및 데이터 로직

필터링 및 제어 임계값 설정

신호 안정화를 위해 5초 이동 평균 필터를 적용했으며, 로봇 관절 J1을 제어하는 규칙은 다음과 같습니다:

  • Threshold: 0.4 (정규화된 High β Power)
  • 집중 (Move +15°): Avg(High β Power) ≥ 0.4
  • 이완 (Move -15°): Avg(High β Power) ≤ 0.2

데이터 처리 파이프라인 (Data Processing Pipeline)

RAW SIGNAL Brainlink (EEG)
FILTERING 5s Moving Avg.
LOGIC Threshold Check
ROBOT CONTROL Spike Prime J1

Python 시뮬레이션 프로그램 검증

로봇 연동 전, 개발한 데이터 처리 로직의 유효성을 Python Plotly 대시보드를 통해 성공적으로 검증했습니다.

시뮬레이션 프로그램 실행 (Streamlit App)
